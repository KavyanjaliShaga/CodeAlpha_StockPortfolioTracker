# Stock Portfolio Tracker

This is a Python-based Stock Portfolio Tracker developed as part of the CodeAlpha Internship.

## Features
- Add stock name and quantity
- Calculate total portfolio value
- Store portfolio details in a file

## Files Included
- stock_tracker.py – Main Python program
- portfolio.txt – Stores portfolio data

## How to Run
1. Make sure Python is installed
2. Run the file using:
   python stock_tracker.py
